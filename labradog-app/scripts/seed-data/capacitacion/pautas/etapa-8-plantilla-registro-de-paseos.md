# Pauta Etapa 8 — Plantilla de registro de paseos

**Uso:** pauta de la evaluación práctica de la etapa 8 (checklist real en terreno + revisión de registros). En producción, el registro vive en la plataforma (Epic 4: checklist pre-paseo, registro post-paseo y reporte al tutor); esta plantilla define el contenido mínimo del método.

**Objetivo:** dejar evidencia objetiva del servicio, registrar el estado del perro y respaldar decisiones profesionales.

## 1. Datos generales del servicio

- Fecha, hora inicio, hora término, duración real (minutos)
- Nombre del paseador/a, nombre del perro, tutor/a

## 2. Condiciones del entorno

- **Clima:** soleado / nublado / lluvioso / calor / frío
- **Entorno predominante:** residencial tranquilo / urbano moderado / urbano intenso / parque / zona con obras o ruido

🔎 *Observaciones del entorno.*

## 3. Estado del perro (lectura emocional)

- **Estado general durante el paseo:** calmo / excitado regulable / estrés leve / estrés alto / ansiedad o pánico
- **Señales observadas:** bostezos repetidos / lamer hocico / fijación visual / rigidez corporal / jadeo excesivo / evitación / ninguna relevante

🔎 *Descripción breve.*

## 4. Manejo y ajustes realizados

- **Tipo de correa utilizada:** corta / media / media–larga
- **Ajustes durante el paseo:** acorté correa / cambié ruta / reduje ritmo / aumenté distancia de estímulos / realicé pausas / terminé antes el paseo

🔎 *Motivo de los ajustes.*

## 5. Interacciones y estímulos

- **Perros:** no hubo / a distancia / evitados / estímulo relevante
- **Personas / niños:** sin contacto / intento de acercamiento / manejo preventivo
- **Otros estímulos** (bicicletas, ruidos, etc.)

## 6. Incidentes / emergencias

(Marcar solo si ocurrió algo relevante)

- Escape / intento de pelea / mordida / lesión / crisis de pánico / ninguno

🔴 *Descripción objetiva del incidente.*
🔴 *Acciones tomadas.*

## 7. Condición final del perro

Al finalizar el paseo, el perro se encontraba: tranquilo / levemente cansado / muy cansado / estresado / requiere seguimiento

🔎 *Observaciones finales.*

## 8. Comunicación al tutor

- ☐ Reporte enviado
- ☐ Incidente informado
- ☐ Recomendación entregada

*Recomendación profesional (si aplica).*

## 9. Firma y cierre

Firma del paseador/a y fecha.
